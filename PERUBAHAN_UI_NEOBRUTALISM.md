# Perubahan UI Neobrutalism - Kopi Nusantara

Project sudah diubah ke gaya **neobrutalism UI** dengan ciri visual:

- Border hitam tebal di card, button, input, navbar, QR card, dan section utama.
- Hard shadow/kotak tegas menggantikan shadow lembut.
- Warna kontras: kuning, pink, biru, hijau, orange, hitam, dan putih.
- Typography lebih bold menggunakan font heading `Space Grotesk`.
- Hero section dibuat lebih kuat dengan headline besar, sticker label, CTA tebal, dan image card brutal.
- Menu card dibuat lebih berani, tidak monoton, dan lebih cocok untuk portfolio.
- Gallery, CTA banner, footer, admin login, QR simulator, customer order, dan dashboard admin sudah mengikuti arah visual neobrutalism.
- Logic order, QR, localStorage, admin, menu, dan tracking pesanan tidak diubah.

## Cara Menjalankan

```bash
npm install
npm run dev
```

## Cara Build

```bash
npm run build
```

Hasil build ada di folder `dist` dan siap dipakai untuk deploy Netlify/Vercel.
