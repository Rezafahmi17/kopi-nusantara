
## Update Database MySQL/phpMyAdmin

Versi ini sudah memakai database asli MySQL yang bisa diakses lewat phpMyAdmin. Import file SQL berikut sebelum menjalankan aplikasi:

```txt
Nusantara/database/kopi_nusantara.sql
```

Panduan lengkap ada di:

```txt
Nusantara/PANDUAN_DATABASE_PHPMYADMIN.md
```
